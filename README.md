# The Metal Scene 🤘

An interactive, mobile-responsive festival roster tracker and custom clashfinder companion app.

## 🚀 Key Features
* **Live Lineup Streams:** Search and instantly filter through massive festival rosters on every single keystroke.
* **Genre Pill Filtering:** Dynamically isolate specific music scenes (Metalcore, Hard Rock, Heavy Metal) with quick-toggle badges.
* **Cross-Referenced Tour Engine:** Spot active touring bands instantly with the `✈️ Tour Matches` badge, featuring custom hover tooltips showing regional dates and venues.
* **Personal Clashfinder Matrix:** Star your absolute favorite acts to compile a personalized festival itinerary tracked seamlessly in memory.
* **Interactive Weighting Engine:** Rank your top choices in a sticky compilation dock to calculate live, weighted leaderboard metrics.

## 📱 Mobile Architecture
Built using fluid responsive media query matrices, shifting the desktop sidebar navigation to an elegant horizontal top-dock layout for seamless tracking on phones and tablets.

## 🛠️ Tech Stack
* **Frontend:** Semantic HTML5, Core CSS3 (Custom Properties & Responsive Design)
* **Backend Logic:** Vanilla JavaScript (Async/Await Fetch API Architecture)
* **Database Endpoint:** Relational JSON static matrix (`app_datasource.json`)
